Installation

🐧 Linux (Ubuntu)

  1. Télécharger le fichier .deb ci-dessus.
  2. Ouvrir un terminal dans le dossier de téléchargement.
  3. Exécuter : sudo dpkg -i passwordproject_1.0.0_amd64.deb
  4. Chercher Password Project dans les applications.

🪟 Windows

  1. Télécharger PasswordProject_1.0.0_Setup.exe.
  2. Double-cliquer sur le fichier téléchargé.
  3. Suivre l'assistant d'installation.
  4. Lancer depuis le Bureau ou le Menu Démarrer.
